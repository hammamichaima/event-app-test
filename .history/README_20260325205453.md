# Event App — Test Technique Fullstack

## Structure
- `/web` — Interface web + API REST (Next.js)
- `/mobile` — Application mobile (React Native + Expo)

## Prérequis
- Node.js 18+
- Expo Go (sur téléphone)

## Lancer le backend + web
cd web
npm install
# Créer .env.local avec DATABASE_URL et JWT_SECRET
npm run dev
# → http://localhost:3000

## Lancer le mobile
cd mobile
npm install
# Modifier constants/api.js avec ton IP locale
npx expo start
# Scanner le QR code avec Expo Go
